git安装后使用
1. 设置用户信息
   1. git config --global user.name "My Name"
   2. git config --global user.email "xxx@qq.com"
2. git init 
   1. 初始化仓库，在文件夹中初始化，生成一个.git文件
   
Git命令
git add <文件名> <文件名> // 可以添加多个文件

git commit -m "上传描述" //  使用add添加的文件都可以一次性上传
